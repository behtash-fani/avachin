# نسخه کاربری فارسی آواچین

آواچین v12.15 یک رابط اولیه کاربرمحور برای Windows دارد. این رابط راست‌چین، فارسی و با طراحی الهام‌گرفته از Material و رنگ‌های Dracula ساخته شده است.

## اجرا

روی فایل زیر دوبار کلیک کنید:

```text
Avachin.bat
```

یا در PowerShell اجرا کنید:

```powershell
.\scripts\windows\avachin.bat
```

## بخش‌های برنامه

### خانه و پیش‌نمایش

- نمایش آمادگی `fpcalc` و `ffmpeg`
- نمایش تعداد آهنگ‌های بانک محلی و سهمیه باقی‌مانده AudD
- انتخاب پوشه موسیقی
- اجرای امن Preview با حالت آفلاین پیش‌فرض
- نمایش پیشرفت، گزارش‌ها و جزئیات اجرا
- توقف امن پردازش جداگانه

### بررسی آهنگ‌ها

- نمایش فایل‌های `REVIEW` و `REJECT` از جدیدترین گزارش واقعی
- پخش فایل انتخاب‌شده
- شناسایی آنلاین کنترل‌شده با AcoustID و fallbackهای محافظت‌شده
- ویرایش فارسی نام هنرمند، عنوان و آلبوم
- ثبت نتیجه فقط بعد از تأیید کاربر
- ساخت Backup و Audit و امکان Undo

### نام‌های هنرمند

- تشخیص نام‌های فاصله‌دار یا املایی مشابه
- انتخاب نام اصلی فارسی یا لاتین
- پیش‌نمایش Recordingها، فایل‌های ثبت‌شده و پوشه‌های فعلی
- یکسان‌سازی بانک محلی بدون درخواست اینترنتی
- ثبت عملیات قابل بازگشت

### تاریخچه و بازگشت

- مشاهده همه اصلاحات انسانی
- نمایش دلیل و جزئیات فنی هر عملیات
- Undo عملیات انتخاب‌شده یا آخرین عملیات

## فونت

برنامه به‌ترتیب این فونت‌ها را انتخاب می‌کند:

1. Vazirmatn
2. Vazir
3. Vazir UI
4. Tahoma
5. Segoe UI

برای ظاهر کامل فارسی، فونت Vazirmatn را روی Windows نصب کنید. فایل فونت داخل Repository نگهداری نمی‌شود.

## مرز ایمنی نسخه اولیه

- Organizer فقط در حالت Preview اجرا می‌شود.
- هیچ دکمه‌ای برای جابه‌جایی، تغییر نام یا Retag کردن آرشیو اصلی وجود ندارد.
- شناسایی آنلاین فقط پیشنهاد می‌دهد و بدون تأیید کاربر چیزی یاد نمی‌گیرد.
- اصلاح بانک محلی و Aliasها از مسیر موجود Backup/Audit/Undo انجام می‌شود.
- هیچ GUI به‌صورت مستقیم به SQLite نمی‌نویسد.

Apply واقعی فایل‌ها بعد از عبور از Sandbox Apply، Crash recovery و Rollback فعال خواهد شد.
